# vue 강의정리

#### 개발환경 설정
- chrome 브라우저 설치
- [vs code 설치](https://code.visualstudio.com/) 
  - plugin 설치 : Vetru, TSLint 설치
- node 설치 
  - brew install node
- vue 설치 
  - npm install vue
  - npm install -g vue-cli

![설치확인](./image/설치확인.png)

#### vue project 생성 및 실행
1. vue init webpack-simple vue-todo : vue-todo 프로젝트 생성
2. cd vue-todo : 프로젝트 디렉토리 이동
3. npm i : 프로젝트 설치
4. npm run dev : 실행
